利用Github和Hexo搭建个人博客

1、准备工作

课堂上，扎西尖措学长为我们讲解了个人博客搭建的主要过程，在欣赏了的前辈们丰富多彩的的blog主页后，我也萌生了搭建个人博客的想法。

遇到的问题

首先便是注册Github账号。由于中国IP很难登录Github网站，我尝试了很久都没能进入。

解决方法

我通过寻求同学帮助下载VPN，成功登录并注册Github账号。

2、安装配置hexo

遇到的问题1

安装文本编辑器Typora时，我发现正版软件需要收费，同学给我发来了破解版安装教程（不过我们依旧要支持正版），打开命令行窗口进行安装时一直报错。

解决方法1

我与同学开了一个腾讯会议，共享我的屏幕一起分析问题，最后发现是教程中命令行将一个下划线打成了空格，导致路径错误找不到文件位置。由此让我明白编程学习需要耐心，追求准确，一个符号的失误就会导致结果错误。

遇到的问题2

在更换Hexo主页的主题时，我发现命令行窗口总是最后一项报错说“Deployer not found: git”。

解决方法2

同样我与同学开了一个腾讯会议，发现两人的情况完全一样，但我们没有发现问题的根源，于是我们借助AI工具，询问豆包应如何解决。采用豆包给出的方法，在命令行窗口里输入”npm install hexo-deployer-git –save“，也就是安装了Hexo博客框架的hexo-deployer-git插件，将Hexo生成的静态网页部署到Git仓库，实现博客的上线发布。

3、撰写，本地预览

遇到的问题

安装好了文本编辑器Typora后，打开它却发现我不知从何下手。

解决方法

通过询问同学、使用AI工具，我学习了基本的markdown语法并试着运用，在边学边写中我逐渐掌握markdown语法，也成功编辑了我的第一篇博客。

4、部署博客到Github

遇到的问题

在Git Bash中复制教程中给出的命令却始终不成功。

解决方法

在与同学交流过后得知，在Git Bash 中不能使用”ctrl+V”进行复制，而应该单击右键点击Paste进行复制。
